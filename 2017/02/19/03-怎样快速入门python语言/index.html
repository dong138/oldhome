怎样快速入门python语言

2017/02/19 Python

1. 你为啥要学习python

python这门语言的岁数估计比大多数的学习者都要大，因为它是1989年被发明粗来的的，你算算是不是大很多~~~~

学习python的动机估计有这么几类：

  • 之前搞的其他语言，比如：C语言、C++，现在需要一门快捷、方便、不需要太多底层的工具
  • 之前搞得不是软件这行，比如：之前是搞文学的，搞体育的，想通过一门简洁的语言进入到软件行业
  • …省略…

不管学习python的理由是什么，坚持到底，才能真正的学习好python，否则 还是该干啥干啥 别在这浪费时间、精力。。。

2.6 类和对象

以我之前搞C语言为例，根部不会有类和对象这个概念，即没有这个东西 C语言一样很强大

但是，语言和语言之间有关联也有区别，用python来开发程序时，为了更快速更加灵活的对程序的数据进行处理，python支持类，当然了也就有了对象

想要搞懂这个知识点，一定需要前面说到的基本类型和函数的相关知识（如果还没有通过，还请绕回到原处）

基本的知识：

  1. 先了解类
  2. 了解对象
  3. 编写最基本的类
  4. 生成这个最基本的类的实例
  5. 调用对象的方法
  6. 调用对象的属性
  7. 加以重复上面的过程

找个较大的程序，体会恍然大悟明白面向对象和面向函数不同的感觉，推荐

  • 老王开枪的例子
  • 飞机大战游戏的例子
